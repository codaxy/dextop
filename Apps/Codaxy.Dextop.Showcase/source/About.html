Dextop

Dextop is RIA open-source framework based on .NET/Mono and Sencha Ext JS.

Checkout Dextop source code at GitHub.

Browse Dextop's documentation.

Ask questions about Dextop at StackOverflow (just mark them with [extjs] tag).

Find out more about Codaxy at www.codaxy.com.
